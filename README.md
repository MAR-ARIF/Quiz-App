## ğŸ’¡ How It Works

1. User clicks **Start Quiz** button.
2. The app fetches questions from the Open Trivia Database API.
3. The first question and shuffled options are displayed.
4. User selects an answer and clicks **Next** to move to the next question.
5. This repeats until all questions are answered.
6. At the end, the app displays the user's **score**.
